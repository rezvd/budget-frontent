# Budget Frontend

Минималистичный pet project на React для ведения личного бюджета.

## Режим проекта

Сейчас проект работает в режиме `read-only`:
- чтение операций из Google Sheets
- отображение первых 5 строк в сыром виде для быстрой проверки подключения
- запись в Google Sheets отключена

## Запуск

```bash
pnpm install
pnpm dev
```

## Переменные окружения

Создай `.env` по примеру `.env.example`:

```env
VITE_GOOGLE_SHEET_ID=
VITE_GOOGLE_SHEET_RANGE=logs!A:F
VITE_GOOGLE_SHEETS_API_KEY=
```

Ожидаемые колонки диапазона `A:F`:
- `A`: дата
- `B`: день
- `C`: сумма
- `D`: магазин
- `E`: категория
- `F`: комментарий

## Как проверить чтение

1. Заполни `.env`
2. Запусти `pnpm dev`
3. Нажми кнопку `Синхронизировать из Google Sheets`
4. Убедись, что в блоке проверки появились первые 5 записей

Если получаешь ошибку `403/404`, проверь:
- правильность `sheet id`
- корректность `range`
- включен ли `Google Sheets API`
- доступ к таблице для чтения
