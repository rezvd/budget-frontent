# FSD Structure

- `app` — инициализация приложения и глобальные стили
- `pages` — страницы (в проекте пока одна: `budget`)
- `widgets` — крупные блоки страницы
- `features` — пользовательские действия (например, создание операции)
- `entities` — доменные сущности (операция бюджета, API, локальное хранилище)
- `shared` — переиспользуемые утилиты

Текущий проект использует базовый уровень FSD без `processes`.
